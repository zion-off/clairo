# clairo

Terminal dashboard for GitHub PRs and Jira tickets.

## Requirements

- Node.js 18+
- [GitHub CLI](https://cli.github.com/) (`gh`) installed and authenticated

## Install

```bash
pnpm install
pnpm build
```

## Run

```bash
pnpm start
```

Or for development:

```bash
pnpm dev
```

## Keyboard

- `1-4` - Switch between boxes
- `j/k` - Navigate lists
- `Enter` - Select
- `Esc` - Cancel
- `Ctrl+C` - Quit
